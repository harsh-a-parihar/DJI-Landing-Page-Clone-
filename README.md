# DJI Clone Project
-------------------

## Introduction
This project is a clone of the DJI website. The purpose of this project is to practice and demonstrate the skills I have learned in HTML, CSS and Bootstrap. The project is a static website with no functionality. The website is responsive and can be viewed on any device.

## Technologies
- HTML
- CSS
- Bootstrap

## Launch
To run this project, simply open the `index.html` file in your browser.
